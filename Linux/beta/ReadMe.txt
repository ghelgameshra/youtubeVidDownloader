                    YOUTUBE VIDEO DOWNLOADER

    Program ini dibuat menggunakan bahasa c++
yang dikombinasikan dengan menggunakan youtube-dl
youtube-dl adalah sebuah scriptberbasis cli
yang digunakan untuk mendownload video youtube dari youtube link
program ini masih dalam tahap pengembangan,
karena program ini dibuat hanya untuk tujuan pendidikan

download manual :
https://github.com/ytdl-org/youtube-dl.git
or,
https://ytdl-org.github.io/youtube-dl/download.html

[!] masalah utama dalam youtube-dl adalah kecepatan download yang
masih lambat, kecepatan maksimal yang didapat saat program di test
adalah 70kb/s.
